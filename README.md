## Overview
This repository contains the manuscript, source files, and visual guides for the paper **Null-Directional Kinematics and the Statistical Structure of Spacetime: Clock Functionals, Transport Geometry, and Directional Support**.

This work investigates a minimal kinematic framework in which physical structure is organized by a normalized ensemble of invariant-speed null directions. It demonstrates that several foundational structures of relativistic and quantum physics do not need to be postulated independently, but can be mathematically recovered as emergent statistical functionals of this invariant null-directional substrate.

## The Null-Directional Statistical Theorem
The core result of this paper establishes a direct mapping between the statistical properties of the local null ensemble and the observed macroscopic laws of physics:

* **Special Relativity (Time Dilation):** The Lorentz factor emerges exactly from the symmetric statistical overlap (Bhattacharyya coefficient) of opposing directional weights.
* **General Relativity (Gravity):** The Newtonian potential arises from the spatial transport of the ensemble's first moment, while the Einstein Field Equations emerge as the unique leading-order covariant closure of the directional strain (relative entropy).
* **Quantum Mechanics (Uncertainty):** The Heisenberg uncertainty bound is recovered as the Fourier duality between spatial localization and directional (momentum) resolution.
* **Gauge Structure:** Geometric connection fields (Yang-Mills structure) are recovered as the kinematically mandatory structures required for local comparison of the internal frame redundancy inherent in the spinor representation of directional amplitudes.

## Falsifiable Prediction: The Decoherence Bound
Unlike purely interpretive frameworks, this statistical ontology predicts a **Planck-anchored stochastic transport template**. The finite informational throughput of the null substrate implies an irreducible, mass-amplified (momentum-quadratic) decoherence floor ($\Gamma_\phi \propto p_{\rm arm}^2$). This provides a concrete experimental discriminator between deterministic tidal shear (baseline GR+QM) and stochastic directional jitter, testable in next-generation macroscopic matter-wave interferometers.

## Repository Structure
* `/paper`: Contains the LaTeX source code (`.tex`) and the compiled `.pdf` of the main manuscript.
* `/visual guide`: Contains supplementary visual and educational materials explaining the directional geometric mechanisms of the theory.
* `/code`: Python scripts (Jupyter Notebooks) used exclusively to generate the diagrams and geometric plots for the visual guide. 
* `/figures`: The exported high-resolution image files generated by the code.

## Recent Updates (Version 6)
